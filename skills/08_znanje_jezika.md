# 🌐 Znanje Jezika — Prijevod, Učenje, Pisanje

## PRIMARNA PRAVILA
- **Hermes uvijek komunicira na BOSANSKOM**
- Prijevodi se rade odmah bez pitanja
- Pisanje/korekcija teksta = odmah piši, ne samo savjeti

## Bosanski Jezik

### Gramatika — Ključne napomene
- **Padeži**: nominativ, genitiv, dativ, akuzativ, vokativ, instrumental, lokativ
- **Glagolski vidovi**: svršeni (napisati) / nesvršeni (pisati)
- **Futur I**: pisat ću / ja ću pisati
- **Kondicional**: pisao bih
- **IJekavica**: mlijeko, dijete, vrijeme (standard u FBiH)
- **Ekavica**: mleko, dete, vreme (srpski standard)

### Česte greške
```
❌ "Ja sam ušao u auto" → ✅ "Ušao sam u auto"
❌ "Ovo je moj novi auto kojeg sam kupio" → ✅ "...koji sam kupio"
❌ "Hvala ti na pomoći" (formalno) → ✅ "Hvala Vam na pomoći"
```

## Engleski Jezik

### Nivoi: A1 do C2
- **A1-A2**: osnovna komunikacija
- **B1-B2**: svakodnevna upotreba, posao
- **C1-C2**: akademski, književni nivo

### Gramatički naglasci
```
Present Perfect vs Past Simple:
"I have been to Paris" (iskustvo, nema vremena)
"I went to Paris last year" (specifično vrijeme)

Conditionals:
Type 1: If it rains, I will stay home.
Type 2: If it rained, I would stay home.
Type 3: If it had rained, I would have stayed home.
```

### Poslovni Engleski
- **Email**: Dear Mr./Ms., I am writing to...
- **Prezentacija**: Good morning everyone, today I'll be presenting...
- **Pregovaranje**: We'd be willing to consider..., That's not quite what we had in mind...

## Prijevodi — Odmah Uradi

### Workflow za prijevod
```
1. Identifikuj jezik izvora
2. Razumi kontekst (formalni/neformalni, tehnički/književni)
3. Prevedi sa prilagodbom stila
4. Provjeri gramatiku i prirodnost
5. Ponudi alternativne prijevode ako postoje

Alati:
- web_search("[fraza] translation") za idiome
- web_search("[termin] [jezik] definicija") za tehničke termine
```

### Formati prijevoda
```markdown
**Original (EN):**
The quick brown fox jumps over the lazy dog.

**Prijevod (BS):**
Brza smeđa lisica skače preko lijenog psa.

**Napomene:**
- "lazy" → "lijen" (pridev u nominativu)
```

## Ostali Jezici — Osnovno Znanje

### Njemački
- Član: der/die/das (m/f/n)
- Glagol na kraju zavisne rečenice
- Složenice: Donaudampfschiffahrtsgesellschaftskapitän

### Francuski
- Nasal vokali: an, en, in, on, un
- Spajanje: liaison "les enfants" = /lez ɑ̃fɑ̃/
- Subjonctif za želje i sumnju

### Arapski
- Pisanje zdesna nalijevo
- Korjenski sistem (3-4 slova)
- Fusha (klasični) vs. darija (govorni)

### Turski
- Vokalska harmonija
- Aglutinativan jezik
- SOV red riječi

## Pisanje i Korekcija Teksta

### Korekcija — Odmah
```
Korisnik pošalje tekst → Ispravi SVE greške → Objasni promjene
```

### Stilski savjeti
- **Kratke rečenice** za čitljivost
- **Aktiv** umjesto pasiva gdje moguće
- **Konkretno** umjesto apstraktnog
- **Bez žargona** ako nema razloga

### Pisanje od nule
```
Korisnik: "Napiši mi email šefu..."
→ Odmah napiši kompletan email
→ Ponudi formalnu i neformalnu verziju
→ write_file("email_[tema].txt", sadrzaj)
```
